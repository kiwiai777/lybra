# Role Catalog Directory

This directory is reserved for future Lybra role template catalog files.

AIPOS-97 defines the protocol for the catalog in `0_control_plane/agents/heterogeneous_agent_role_catalog_protocol.md`.

The catalog is the Template Layer. It is vendor-neutral and uses functional role names.

The existing agent capability registry remains the Instance Layer. Concrete agent instances, model choices, hosts, runtime profiles, and availability do not belong in template names.

AIPOS-97 does not create concrete role template YAML files. Future template files may be added only through a separate Owner-approved implementation task and independent audit.
