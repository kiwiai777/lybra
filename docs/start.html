INSTALL

Install

Requirements: Node.js 18+ (and Python 3 for the local Board and MCP server).

npm install -g lybra

Package name: lybra.

CLI entry point
lybra --help
python3 -m tools.aipos_cli.aipos_cli --help

Source checkout uses the Python module entry point above.

Quick start

Start the local Board (default port 7117):

python3 web/board/app.py --host 127.0.0.1 --port 7117 --repo-root /path/to/your/workspace

Then open:

http://127.0.0.1:7117

Access from another machine over SSH:

ssh -N -L 7117:127.0.0.1:7117 user@your-host

Connect an agent over MCP (HTTP/SSE default port 7118):

python3 -m tools.mcp_server serve-http --host 127.0.0.1 --port 7118
http://127.0.0.1:7118
