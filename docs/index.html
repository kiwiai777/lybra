CORE PRINCIPLES

Four core principles

Files as source of truth

State lives in durable files, not compressed conversation memory — state outlives the model.

The owner holds every gate

Architecture, risk and scope decisions are non-delegable.

Independent audit, enforced

An executor can never audit its own work.

Agents come to Lybra

MCP-native; no autonomous runtime, no heartbeat polling.
